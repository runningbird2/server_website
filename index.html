About FabricCraft

We run a survival server that rewards resilience, clever builds, and diplomacy. The world does not reset and there is no land claiming, so every base is defended by players, not plugins. We run Fabric (not Paper/Spigot), so vanilla mechanics stay intact for farms and contraptions.

What makes it different

We keep the rules tight and the world alive. If you can build it, defend it, or negotiate for it, it is yours to keep. The map evolves through conflict, collaboration, and clever design.

Rules

Keep it fierce, not toxic. These rules protect the community while preserving the risk.

No hacked clients

Any client or mod that grants unfair advantages is banned. Freecam is allowed.

Griefing allowed, spawn is protected

Raids and destruction are part of the game, but no spawn griefing for the sake of griefing (lavacasts included).

No exploits or server sabotage

Dupes, lag machines, and crash attempts that destabilize the server are removed.

No doxxing or targeted harassment

Threats, hate speech, and real-world harassment are banned.

Respect staff in emergencies

Staff only steps in to keep the world stable. Disputes can be reviewed later.
